# Необходимые дополнительные поля для DLE


## Основные поля рецепта (7 полей)

### 1. cook_time
- **Название:** Время приготовления
- **Тип поля:** Текстовое поле
- **Использование:** Время приготовления в минутах (только число)
- **Пример:** 40

### 2. calories
- **Название:** Калорийность
- **Тип поля:** Текстовое поле
- **Использование:** Калорийность в ккал (только число)
- **Пример:** 285

### 3. servings
- **Название:** Количество порций
- **Тип поля:** Текстовое поле
- **Использование:** Количество порций (только число)
- **Пример:** 4

### 4. difficulty
- **Название:** Сложность
- **Тип поля:** Выпадающий список
- **Варианты:** Легко, Средне, Сложно
- **Использование:** Уровень сложности приготовления

### 5. ingredients
- **Название:** Ингредиенты
- **Тип поля:** HTML-редактор или текстовая область
- **Использование:** Список ингредиентов с количеством
- **Пример:**
```html
<ul>
<li>Картофель - 500г</li>
<li>Лук - 2 шт</li>
<li>Морковь - 1 шт</li>
</ul>
```

### 6. recipe_steps
- **Название:** Шаги приготовления
- **Тип поля:** HTML-редактор или текстовая область
- **Использование:** Пошаговая инструкция приготовления
- **Пример:**
```html
<ol>
<li>Очистите и нарежьте картофель кубиками</li>
<li>Обжарьте лук до золотистого цвета</li>
<li>Добавьте морковь и тушите 5 минут</li>
</ol>
```

### 7. video
- **Название:** Видео рецепта
- **Тип поля:** Текстовое поле
- **Использование:** Ссылка на видео (YouTube, VK и т.д.)
- **Пример:** https://www.youtube.com/watch?v=...

## Категории рецептов

**ВАЖНО:** Для отображения цветных иконок категорий в fullstory.tpl используется стандартный функционал DLE через теги `[catlist=ID]`.


1. **Первые блюда** (ID: 1) - красный цвет
2. **Вторые блюда** (ID: 2) - оранжевый цвет
3. **Гарниры** (ID: 3) - фиолетовый цвет
4. **Салаты** (ID: 4) - зелёный цвет
5. **Закуски** (ID: 5) - розовый цвет
6. **Десерты** (ID: 6) - жёлтый цвет
7. **Напитки** (ID: 7) - голубой цвет
8. **Детям** (ID: 8) - бирюзовый цвет

**Если ID категорий отличаются**, отредактируйте файл `fullstory.tpl` и замените `[catlist=1]` на актуальные ID ваших категорий.

## Примечания

1. Все текстовые поля с числами (cook_time, calories, servings) должны содержать только числа без единиц измерения - единицы добавляются автоматически в шаблоне.

2. Для полей ingredients и recipe_steps рекомендуется использовать HTML-редактор для форматирования списков.

3. Поле video необязательное - если оно не заполнено, значок видео не отображается на карточке рецепта.

4. Для иконок категорий используется стандартный тег DLE `[catlist=ID]`, дополнительные поля создавать не нужно.

## Установка в DLE

1. Войдите в админ-панель DLE
2. Перейдите в раздел "Настройки" → "Поля для пользователей/новостей"
3. Создайте все 7 перечисленных полей согласно спецификации
4. Создайте необходимые категории
5. При добавлении новых рецептов заполняйте эти поля для корректного отображения
